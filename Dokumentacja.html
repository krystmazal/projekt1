Dokumentacja Aplikacji Notatnik

Ta aplikacja to notatnik z interfejsem graficznym stworzonym w Pythonie z wykorzystaniem Tkintera oraz bazą danych MySQL. Umożliwia użytkownikowi rejestrację, logowanie oraz zarządzanie swoimi notatkami.

Spis treści

  1. Wymagania
  2. Struktura plików
  3. Uruchamianie aplikacji
  4. Opis funkcjonalności
  5. Klasa Database

1. Wymagania

  • Python 3.x
  • Biblioteka tkinter (wbudowana w Pythona)
  • Biblioteka mysql-connector-python

2. Struktura plików

  • main.py - główny plik aplikacji zawierający logikę Tkinter
  • baza.py - zawiera klasę Database odpowiedzialną za operacje na bazie danych

3. Uruchamianie aplikacji

  1. Zainstaluj wymagane biblioteki: 
    pip install mysql-connector-python
  2. W pliku main.py podaj dane logowania do swojej bazy danych (host, użytkownik, hasło, nazwa bazy).
  3. Uruchom aplikację: 
    python main.py

4. Opis funkcjonalności

Rejestracja

Funkcja rejestracja() pozwala nowym użytkownikom na stworzenie konta. Jeśli użytkownik o podanym loginie już istnieje, zostaje wyświetlony odpowiedni komunikat o błędzie.

Logowanie

Funkcja logowanie() sprawdza poprawność loginu i hasła użytkownika. Po zalogowaniu użytkownik zostaje przeniesiony do swojego notatnika.

Dodawanie i wyświetlanie notatek

Użytkownik może dodawać nowe notatki, które są zapisywane w bazie danych i przypisane do jego konta. Wszystkie notatki użytkownika są wyświetlane na liście.

Usuwanie notatek

Użytkownik może usunąć dowolną ze swoich notatek. Notatka zostaje usunięta z bazy danych.

5. Klasa Database

Klasa Database zarządza połączeniem z bazą danych oraz operacjami CRUD (tworzenie, odczyt, aktualizacja, usuwanie) dla tabel użytkowników i notatek. Poniżej znajduje się opis głównych metod:

  • __init__(self, host, user, password, database) - Inicjalizuje połączenie i tworzy bazę danych oraz tabele.
  • create_tables(self) - Tworzy tabele users i notes.
  • check_user(self, login, password) - Sprawdza, czy użytkownik istnieje w bazie.
  • insert_user(self, login, password) - Dodaje nowego użytkownika.
  • get_user_id(self, login) - Pobiera identyfikator użytkownika na podstawie loginu.
  • select_notes_by_user(self, user_id) - Pobiera wszystkie notatki dla danego użytkownika.
  • insert_note(self, tresc, user_id) - Dodaje nową notatkę.
  • delete_note(self, note_id) - Usuwa notatkę na podstawie identyfikatora.
  • close(self) - Zamyka połączenie z bazą danych.

Podsumowanie

Aplikacja notatnika umożliwia prostą obsługę notatek w środowisku graficznym oraz bezpieczne przechowywanie danych w bazie MySQL. Jest to przyjazne narzędzie do zarządzania krótkimi notatkami, które można łatwo rozszerzyć o dodatkowe funkcjonalności.
